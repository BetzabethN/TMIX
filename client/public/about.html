About Us

This is a project for CodeLabs Summer Internship. This website is based on the project idea of our mentor Jacob Cuomo.

The purpose of this website is to put all the streaming services into one website. So instead of having to go to multiples site like Twitch, Youtube, or DLive, you could find all your favorite streamers in one website.

We made this website using HTMl, CSS, Javascript, and the Twitch APi.
